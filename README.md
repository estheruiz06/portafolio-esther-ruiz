## Comandos principales

- `npm install`: Instala todas las dependencias necesarias que están listadas en `package.json`.

- `npm run dev`: Ejecuta el servidor de desarrollo (con recarga automática al guardar cambios).

- `npm run build`: Genera la versión de producción en `dist/`. Nota: si estás abriendo `dist/index.html` (o usas `npm run preview`), los cambios en `src/` no se ven hasta volver a ejecutar `npm run build`.

- `npm run preview`: Sirve lo que haya en `dist/` para previsualizar la build de producción.

## Rutas

Este proyecto usa `createWebHashHistory`, así que las rutas se navegan con `/#/` (por ejemplo: `http://localhost:5173/#/diseno-editorial`).
